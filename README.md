# ABOUT

**Greetings, Friend** is a web app for sending personalized greeting cards. Pick a card, write your message, and share a link with someone special.

[Try it out here](https://greetings-friend.netlify.app/)

# TECH STACK

- Vanilla HTML, CSS, JavaScript
- Node.js and Express
- MongoDB
- Cloudflare R2
- Transactional email for OTP reset

# FEATURES

- Browse a gallery of greeting card templates
- Customize your card with a personal message using the card editor
- Share a unique link that the recipient can open to view their card
- Create an account, log in, and reset your password via email OTP
